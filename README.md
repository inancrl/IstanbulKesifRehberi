**İstanbul'daki en iyi işletmeleri keşfetmenin akıllı yolu!** Yıldız puanına göre filtreleme yaparak mükemmel mekanları bulun.

## ✨ Öne Çıkan Özellikler
- 🗺️ **Google Maps Entegrasyonu** - Gerçek zamanlı konum ve mekan bilgileri
- ⭐ **Puan Filtreleme** - 1-5 yıldız arası özel sıralama
- 🏷️ **7 Kategori**:
  - ☕ Restoran & Kafe
  - 🛍️ Alışveriş
  - 💆 Sağlık & Güzellik
  - 🛠️ Hizmetler
  - 🎉 Eğlence
  - 🎓 Eğitim
  - 🏨 Konaklama
- 📱 **Tam Duyarlı Tasarım** - Tüm cihazlarda mükemmel görüntüleme
